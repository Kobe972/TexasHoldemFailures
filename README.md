# TexasHoldemFailures
德州扑克2100手牌中犯的错误（由黑桃大师GTO策略提供）

改正后黑桃大师测评由LV5提高到LV9（满分）

1.	翻前KK加注，被一个人跟，翻牌4h4cAd，加注被跟，转牌出5s，下注错误：对手翻牌圈跟注后，牌力不够再次价值下注，更建议过牌控池。
2.	翻前KK加注，被一个人跟，翻牌8d3cAs，加注被跟，转牌出6s，下注错误：对手翻牌圈跟注后，牌力不够再次价值下注，更建议过牌控池。
3.	翻前KK加注，被一个人跟，翻牌AcTd7s，下注错误：牌面有A，我们的牌力不够价值下注，也没有必要转牌诈唬。
4.	翻前KhKc，翻牌出h同花面过牌，转牌出9h，过牌错误，应该下注。
5.	翻前KK加注，被一个人跟，翻牌AsTs2d，都过牌，转牌出5c，对手下注，跟注，河牌出7d，对手过牌，下注错误：下注通常没什么价值，还有被加注的风险。
6.	前位持有KK，翻牌转牌全过牌，河牌牌面7cAcJc4s8h，下注错误：对手连续过牌，范围不强，但是河牌让他的一些组合实现了反超。（如果河牌出个4什么的，则可以下注）
7.	持有KK，牌面有A，对手转牌和河牌持续下注，河牌不应跟注：对手持续下注，显示了一定的实力，我们领先的机会不大，范围里有足够多的Ax可以防守，不建议继续跟注
8.	持有KK，翻牌圈对手下注，我跟注，转牌击中暗三，下注错误：没有足够理由反主动下注。
9.	持有KK，翻牌转牌圈全部过牌，河牌牌面Jc7cAc4s4h，过牌错误：对手连续过牌，范围不强，河牌也很难帮到他，我们有较大机会领先，应该考虑薄价值下注。过牌可能损失价值。
10.	持有AsKh，翻前加注，一家跟注，翻牌圈和转牌圈过牌，河牌牌面9s5c8c2d5d，对手过牌，下注错误：下注没有什么价值，也很难让对手弃掉有对子的牌，我们还有领先的可能，过牌摊牌就好。
11.	持有AsKc，翻前加注，一家跟注，翻牌圈过牌，转牌对方下注，跟注，河牌牌面8d9c6cAh4h，对手先过牌，过牌错误：对方过牌后，我们通常领先，随后过牌容易损失价值。
12.	AK翻牌卡顺听牌下注半诈唬被跟注后，转牌牌面ThJs3d4d，不应后位过牌，应该继续下注到满池。
13.	持有AdKh，翻牌牌面JsTh3d，过牌，转牌出4d，前位过牌错误：我们AK有坚果卡顺听牌和两高张，有不错的底池权益和弃牌率，更建议下注诈唬。
14.	持有AdKh，翻牌牌面JsTh3d，翻前加注跟一家，翻牌圈后位过牌错误：应该下注半诈唬。
15.	持有AhKs，翻前加注跟一家，翻牌牌面4s2d8d，过牌被下注1bb，弃牌错误：对手下注尺度小，范围较宽，我们的牌有可能领先，底池赔率很好，直接弃牌太紧。
16.	持有AdKh，翻前加注跟一家，翻牌出9d4dJd，过牌，转牌出6h，前位过牌错误：对手翻牌圈后位过牌后，我们在转牌下注有一定的弃牌率。而且阻断了坚果同花，被加注的可能性很小，更建议下注半诈唬。
17.	AK翻前加注跟一家，翻牌出Tc2d6s，加注被跟，转牌出Jh，对方看牌，看牌错误：对手翻牌圈只是跟注，范围里没有多少很强的牌，转牌也很难提升牌力。我们的AK有坚果卡顺听牌和两高张，有不错的底池权益和弃牌率，更建议继续下注对对手施加压力。
18.	AK翻牌遇到干燥牌面在多人底池不要下注半诈唬。
19.	AdKh翻前加注跟一家，翻牌圈出9s6s8d，过牌，转牌出3h，下注错误：这里不应该半诈唬，下注很难打走有对子的牌。
20.	AK翻前加注跟一家，翻牌出2sQsTc，过牌，转牌出3c，前位过牌错误，有卡顺听牌应该下注半诈唬。
21.	AcKd，翻前加注跟一家，翻牌出6cJdQh，过牌错误：我们有范围优势和坚果优势，应该高频持续下注。其次，有坚果卡顺听牌和亮高张。
22.	AK翻牌圈击不中时的基本玩法：牌面干燥下注半诈唬，牌面潮湿则过牌。
23.	KJo面对UTG1的率先加注，BB应该弃牌。
24.	KJo翻前一家加注，跟注，翻牌出4cTh9s，下注错误：作为翻牌前被动方，不建议反主动下注。
25.	KJo翻前和翻牌圈都率先加注被一家跟，转牌牌面6hTsKh8s，下注错误：转牌让牌面变得更加湿润，继续下注价值有限，有可能遭遇加注陷入被动局面。
26.	KJo翻前一家加注，跟注，翻牌圈Ad5hTc，下注错误：牌面对翻前激进方有利，不建议反主动下注。
27.	KJo在小盲位应该加注或弃牌。
28.	KJo翻前一家加注，跟注，翻牌圈出9sTh4c，对方下注1BB，弃牌错误；转牌出8d，双方过牌，河牌出2c，前位过牌错误：对手转牌圈后位过牌，范围不是很强，我们的手牌没有摊牌价值，更建议下注诈唬。
29.	KJo遇到MP1加注，CO位应该弃牌：面对中位较紧的率先加注范围，我们的牌力偏弱，更容易被统治。
30.	KJo遇到庄位加注，小盲位应该3bet或者弃牌。
31.	KJo在UTG1应该弃牌。
32.	KJo翻牌前面对中位率先加注，BB应该跟注。
33.	KJo翻前下注被一家跟，翻牌圈4h7c9a，下注错误：牌面对我们不利，我们只有两高张和很少后门可能，没有位置的情况下不适合激进游戏。翻牌过牌，对手下注1BB，跟注错误：虽然对手下注尺度很小，但是我们牌力很弱，没什么发展。
34.	KJo翻牌圈遇到2s9h8h这种完全无关的牌，如果对手下注，即使尺度很小也应该弃牌。
35.	KJo翻牌圈中最大的对子，单挑可以跟到底。
36.	KJo如果翻前率先加注了，翻牌圈遇到卡顺听牌应该下注半诈唬。而如果翻前是被动方，即使翻牌中KK、JJ也应该先过牌。
37.	T9s在大盲位遇到一家加注应该跟注：T9s投机性很强，在终结行动的大盲位置，不要轻易弃牌。
38.	AsKs，三次持续下注被一家跟注，河牌3cQsTsKd5d，后位下注错误：对手在这样的牌面连续跟注我们的牌力还不错，河牌的后花出现既有可能被反超，也降低了对手弱于我们的牌力支付我们的概率，因此直接过牌摊牌更好。
39.	TT在CO位翻前加注后，翻牌圈3sAdJs不应主动下注：我们有一些摊牌价值，而且难诈唬走比我们强的中对、顶对，不建议持续下注。
40.	99在UTG2位置翻前主动加注，遇到小盲3bet应该跟注。
41.	76转牌4357，底池225，对手超池全押1935应该弃牌（最好也只能平分底池）
42.	AA持续下注到转牌，牌面Ks2h7d4d，松凶对手超池全押应该跟注：对手的形象很激进，这手牌处于范围的顶端，跟注有不错的底池赔率。
43.	99翻前加注被一家3bet，跟注，翻牌出6h2d2s，对手后位下注应该跟。
44.	76oBB溜进游戏多人底池，翻牌圈8s5dJd，过牌被下注，其余两家跟注，应该弃牌：牌面有同花听牌和更大顺子听牌的可能，我们的净补牌只有三张。
45.	5d6d翻前加注一家跟，翻牌出Qh3d9s，下注被跟，转牌出5h，后位下注错误：牌面有听花可能，且我们是中对子，继续下注很难让更弱牌跟注，也很难让更强牌弃牌
46.	Th6h大盲位，四人底池溜进，翻牌圈全过牌，转牌KhQdJcAd，一家紧凶超池全押，不应跟注：对手可能坚果顺听同花，此时胜率只有百分之40
47.	JcAd，翻前加注两家跟，翻牌圈中位下注错误：听牌程度中等，在多人底池中间位置，建议听牌平衡范围，如果庄位玩家下注，我们则应加注。
48.	9s7s翻前加注一家跟，翻牌Qs6h2d，小注诈唬，被跟注，转牌出Ts，应该超池诈唬：对手跟注了我们的小尺度下注，范围通常比较封顶，而我们上不封顶，且有顺子和同花听牌。我们的手牌毫无摊牌价值，但是听牌强度很高。
49.	面对后位加注，小盲3bet范围比较宽，不用过度弃牌
50.	如果翻牌圈湿润，但湿润的刚好是我们的听牌/击中的牌，可以诈唬
51.	Ad6d翻牌圈出7h3dAs，不应该在多人底池前位主动下注(中等强度的成牌)
52.	KJo翻牌圈6s5s5c，对手小尺度持续下注后应该跟注，因为有两高张和后门同花听牌
53.	KhQo，翻前加注三家跟，翻牌圈3JTh，不应下注，因为多人底池里净补牌可能是0。一家下注后两家跟注，应该弃牌，可能已经死听。
54.	如果翻前被动，翻牌圈遇到强听牌，过牌后对手后位持续下注，应该加注施加压力，因为对手持续下注范围很广
55.	多人底池且玩家水平不高时，强听牌赔率略微不够可以跟小注，因为有很高的隐含赔率
56.	对手范围有两对这样的牌时，如果想用更强牌诈唬，应该下大注。
57.	9s8s翻前被加注，跟注进多人底池，翻牌出QdKs4s，一家下注，应该跟注，不应该弃牌。
58.	听牌隐秘程度会影响隐含赔率。
59.	翻前AcAd，加注被3bet，进行4bet被一家跟，翻牌圈出7d2cJs，儍凶对手下大注，加注到1/2底池错误：牌面干燥，对上儍凶对手，利用位置慢打是最好的选择。转牌出Kh，对手下注2/3，后手2630，底池变成7700，跟注错误：对手已经彻底套池了，可以全押收网了。
60.	多人底池倾向于快打，即使有一个大鱼玩家。
61.	6c7c，翻前跟了加注，翻牌出5d2h8c，过牌后对方CO位下注，应该加注：两头顺+后门听牌，牌面干燥对方连续下注范围宽且诈唬组合多。
62.	AA遇到3bet，如果都是差玩家，不用4bet。
63.	转牌圈我们是葫芦牌面有同花，底池很大，后手筹码小的情况下，如果对手是儍凶玩家，建议过牌示弱。
64.	A8s在大盲位，就剩一个大鱼小盲位跟的时候应该加注：牌力领先鱼玩家的溜进范围，有位置，为了获取更多的价值。
65.	88翻前加注被一家跟，翻牌圈下注被跟，转牌牌面Qc8h5h7h，前位过牌错误：虽然牌面出现了同花，但是我们仍然领先对手的大部分范围。下注可以获取价值，也可以避免被免费反超，还可以平衡范围。
66.	AKo翻前3bet被一家跟，转牌出48KQo，傻凶下注，加注错误：虽然出现了一些听牌，但是对手诈唬很多，加注往往没什么价值。
67.	KdKc翻前3bet被一家跟，翻牌出Td3c3d，被傻凶下注，应该跟注慢打：我们阻断了对手一些同花听牌组合，且对手诈唬很多。
68.	碰到松弱对手，超强牌河牌可以全押。
69.	中明三条小踢脚，儍凶对手持续下注，河牌跟注即可，因为持续下注范围比较极化。
70.	Axs这样的投机牌在大盲位遇到鱼玩家，翻前应该加注。
71.	KJo翻前加注三家跟，翻牌圈出AcQcTs，下注一家弃牌，一家跟注，一家加注。跟注错误：牌面潮湿，多人底池有人跟注有人加注已经显示了很强的牌力，应该加注。
72.	翻前76s加注一人跟，翻牌圈只中中对，应该持续下注保护，否则容易被反超。
73.	KJs单挑底池对大鱼，持续下注到转牌圈牌面Ts9hQs8h，对手下注，跟注错误：牌面非常潮湿，我们的牌非常强，应该加注获取价值。
74.	明三条干燥牌面河牌遇到儍凶下注，应该加注获取价值。
75.	66四人底池，翻牌圈8c8d6d，加注被紧弱对手3bet，其他玩家弃牌，应该再加注：对手很被动，再加注通常代表持有明三条，跟注慢打容易损失价值。
76.	4c5c翻前3bet被紧弱玩家跟，翻牌圈出Ac5d5h，过牌错误：Axx的公对牌面虽然干燥，但如果对手有A，我们很难让他弃牌，如果对手没A，也很难指望他提升牌力支付我们。在类似牌面上，我们经常会持续下注诈唬，所以有强牌也应该价值下注。
77.	33溜进，翻牌圈6Q3o三个玩家过牌，过牌错误：虽然牌面干燥，但我们的手牌没有阻断顶对组合，也没有阻断听牌组合，面对多个对手，建议下注。
78.	97s溜进翻牌出8hJdTd，被紧弱对手3bet后应该再加注：牌面很湿润，对手跟注范围很强
79.	面对很凶的对手，可以慢打强牌。对手持续下注时，范围比较极化，到河牌圈不要轻易加注。
80.	翻牌圈如果有坚果且牌面潮湿，持续下注后被加注可以再加注(注意“且牌面潮湿”)。
81.	超对转牌遇到干燥牌面也应该持续下注。
82.	Ac8h游戏三人底池，翻牌圈出5cQc6c，两家过牌被紧弱玩家下注到满池，跟注错误：对手很可能有同花，对手的草花阻断了听牌，且隐含赔率不高。
83.	T9s在小盲位加注错误：在小盲位加注，需要更多的牌力。建议补齐底池。
84.	KQo在MP2位加注后被松弱BB3bet，跟注错误：对手的形象很被动，挤压范围很强，这手牌很容易被统治，建议弃牌。
85.	TT在小盲遇到CO位加注，大盲松凶，跟注错误：中口袋对子在小盲位，具有不错的游戏价值。面对后位的加注，当很激进的玩家在大盲位时，不建议带头跟注，建议做一个底池大小的3bet。
86.	76s在CO为遇到MP2的加注，庄位紧凶，SB松凶，应该弃牌或者3bet：有激进玩家，跟注很可能看不到翻牌圈，如果跟注需要更强的牌力。
87.	JTo翻前MP1加注错误：牌力很弱，当前位置不足以加注进池，建议弃牌。
88.	紧弱对手溜进后大尺度3bet，在多人池至少是QQ+和AK这样的牌力，QQ/AK对上QQ+和AK的胜率大约是0.4。
89.	5h4h，河牌牌面JsAd3s5c8h，之前被跟注两次，河牌后位应该过牌：对手跟注两次的范围很强，河牌是一张无关牌，我们通常很难诈唬走对手的成牌，而破产的听牌无需诈唬。
90.	JJ加注入池被一家跟，翻牌圈出3hQs6c，后位下注错误：牌面比较干燥，我们有一个被盖帽的口袋对，在单挑底池里，我们的牌力中等并且不容易受到伤害。转牌出As后位下注错误：我们的牌力既没有强到要去价值下注，也没有弱到要诈唬，建议继续过牌，希望得到一个便宜的翻牌。
91.	KsQd加注入池被一家跟，翻牌圈9s6s2h，前位过牌错误：虽然牌面湿润，但我们没有摊牌价值，且我们的手牌阻断了对手的同花组合。两张S是和我们有关，所以可以在因同花面而湿润的牌面上诈唬。
92.	AKs 3bet后被松凶全押5bet，应该弃牌：对手全押的范围很强，需要更好的底池赔率才能跟注。
93.	87s在MP2加注后被BB 3bet，应该弃牌：牌力不够。
94.	44在庄位遇到UTG2加注，此时有效筹码量为跟注量11倍，我们身后还有3个人，建议弃牌：通常来说小对子至少需要跟注量15倍的后手筹码量，才值得跟注买暗三条。
95.	AQo UTG2翻前加注被松弱MP2 3bet，应该弃牌：我们在前位加注，代表的是一个很强的范围，被动玩家对我们做3bet，通常是一个更强的范围，这里我们应该放弃很多AJ杂色甚至是AQ杂色，根据赔率决定要不要用一些口袋对子去买暗三条。
96.	Ac5c加注入池被傻凶跟，翻牌圈4c2c7d，下注诈唬，被加注，跟注错误：我们的组合听牌牌力很强，对手比较激进，加注的范围很广，我们应该再加注争取直接拿下底池。
97.	AJs翻前加注被一家3bet，跟注，翻牌圈出ThAc3s，双方过牌，转牌出7h，双方过牌，河牌出4c，过牌错误：我们是顶对大踢脚，对手转牌圈后位过牌，一方面对手通常不会在河牌圈诈唬，另一方面，我们的牌力大概率领先，建议河牌主动下注。注：本手牌也可转牌圈下注保护，河牌圈持续下注。
